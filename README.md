# Secure URL Shortener & Analytics Platform

A production-ready URL shortening application built with Java, Spring Boot, Spring Security, JWT, and PostgreSQL. The platform allows authenticated users to generate and manage shortened URLs, redirect users to original destinations, and track click analytics through secure REST APIs.

## Features

* JWT-based Authentication & Authorization
* URL Shortening & Redirection
* Click Analytics Tracking
* RESTful API Architecture
* PostgreSQL Database Integration
* Spring Data JPA Persistence Layer
* Layered Architecture (Controller-Service-Repository)
* Production-Ready Spring Security Configuration

## Tech Stack

* Java 21
* Spring Boot
* Spring Security
* JWT
* Spring Data JPA
* PostgreSQL
* Maven
* Lombok
